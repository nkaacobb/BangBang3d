Lighting Playground ▼

Interactive light creation and manipulation

Axes: X (Red) · Y (Green) · Z (Blue)

Camera Controls:

• Left drag: Rotate

• Right drag: Pan

• Wheel: Zoom

Light Controls:

• Click light to select

• Drag to move (XY plane)

• Ctrl+Drag: Move Z axis

• Shift+Drag: Rotate

• Edit properties in panel

• ✔/✖ toggle enables/disables light

Material Controls (Phase 2 & 3):

• Select object to edit material

• Switch material types (Basic, Lambert, PBR, Debug)

• Apply procedural textures (7 types)

• Adjust PBR properties (metallic, roughness)

• Use presets for quick testing

Note: All light types dynamically affect object shading. Move lights around to see real-time lighting changes!
