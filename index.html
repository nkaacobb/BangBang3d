Quick Start

Using BangBang3d in Your Application

Method 1: Import from Web Server

<script type="module">
  import {
    Scene, Mesh, BoxGeometry, BasicMaterial,
    PerspectiveCamera, BangBangRenderer
  } from 'http://127.0.0.1/repo/Projects/Utilities/BangBang3d/src/index.js';

  const canvas = document.getElementById('canvas');
  const renderer = new BangBangRenderer({ canvas, width: 800, height: 600 });
  const scene = new Scene();
  const camera = new PerspectiveCamera(75, 800/600, 0.1, 100);
  camera.position.z = 5;

  const cube = new Mesh(
    new BoxGeometry(1, 1, 1),
    new BasicMaterial({ color: 0xff0000 })
  );
  scene.add(cube);

  function animate() {
    cube.rotation.x += 0.01;
    cube.rotation.y += 0.01;
    renderer.render(scene, camera);
    requestAnimationFrame(animate);
  }
  animate();
</script>

Method 2: Run Local Development Server

cd d:\Repo\Projects\Utilities\BangBang3d
python -m http.server 8000

# Open http://localhost:8000/examples/basic-cube/

Key Features

  • Complete Pipeline: Scene graph, transforms, lighting, rasterization, depth testing
  • Math Library: Vector2/3/4, Matrix4, Quaternion, Euler, Color
  • Materials: BasicMaterial (flat), LambertMaterial (diffuse), DebugMaterial (visualization)
  • Geometry: BoxGeometry, PlaneGeometry with BufferAttribute system
  • Lighting: AmbientLight, DirectionalLight with CPU-based shading
  • Textures: Procedural generation, UV mapping, filtering, wrapping modes
  • Controls: OrbitControls for interactive camera manipulation
  • Debug Tools: Stats monitoring, DebugMaterial visualization modes
